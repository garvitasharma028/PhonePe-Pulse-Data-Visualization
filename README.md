# PhonePe Pulse Data Visualization (2018–2022)

## Project Overview
An interactive Power BI dashboard visualizing PhonePe's 
digital payment trends across India from 2018 to 2022.

## Domain
Fintech

## Tools Used
- Microsoft Excel (Data Cleaning)
- Microsoft Power BI (Dashboard & Visualization)

## Dataset
6 CSV files covering aggregated transactions, 
aggregated users, map-level transactions, map-level 
users, top transactions, and top users across 
36 Indian states and UTs.

## Dashboard Pages
1. Overview — India geo map, KPI cards, Top 10 states, 
   10 interactive dropdown slicers
2. Trends & Insights — Transaction type breakdown, 
   user growth trend, YoY growth rate, device brand 
   analysis, transaction amount trend

## Key Insights
- Total transaction value: ₹121.41 Trillion (2018–2022)
- Telangana leads all states by transaction amount
- Merchant payments = 46% of all transactions
- Xiaomi is the top device brand among PhonePe users
- Highest YoY growth: 2019 (2.9x over 2018)
- Registered users grew consistently every quarter
